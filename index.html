¿Qué es Wollok Game?

Wollok Game es una herramienta nativa del lenguaje de programación Wollok que permite crear juegos de manera fácil. De esta forma es posible que alumnos de cualquier nivel codifiquen parte de un juego y luego prueben su código jugando. Así buscamos incentivar a la sociedad a interesarse en la programación, y facilitarles la comprensión de los conceptos ofreciendo una interfaz visual y controlable.

El docente deberá comenzar codificando el juego, pero dejando algunas funcionalidades sin hacer para que sean completadas por el alumno. Es muy importante tener siempre en cuenta qué concepto se busca practicar y no perderse en lo divertido que es hacer juegos!

¿Cómo descargarlo?

Antes que nada, es necesario tener Wollok instalado, el cual se puede descargar desde la página oficial.

  • tar.gz

¿Cómo jugar?

El objetivo del juego es ayudar al grajero Orson a salir de su granja. Para ello necesita comprar un tractor. El problema es que el tractor es muy caro y el granjero necesitará cosechar muchos cultivos para lograrlo.

Correr el juego

Para correr el juego basta con ubicar el archivo llamado granja-de-orson-game.wpgm dentro de la carpeta src y ejecutar click derecho > Run As > Wollok Program.

Si vemos dentro del archivo encontraremos un programa Wollok con un comentario explicando los controles y objetivo del juego.

Una vez que el programa se haya ejecutado se abrirá una nueva ventana con nuestro juego, así que a jugar.

El alumno avanzará hacia al objetivo del juego hasta que surja un error...

.. y tenga que resolverlo para continuar.

En el origen del problema se encuentra una explicación acerca de lo que debería hacer el juego. El alumno codificará dichos requerimientos y volverá a jugar para probarlo.

A medida que avance, el juego liberará nuevos lugares y dirá nuevas pistas.

Una vez que se haya completado el juego y ganado, éste terminará y se cerrará solo.
