    ¿Cómo jugar?

    El objetivo del juego es ayudar al grajero Orson a salir de su granja. Para ello necesita comprar un tractor. El problema es que el tractor es muy caro y el granjero necesitará cosechar muchos cultivos para lograrlo.

    Correr el juego

    Para correr el juego basta con ubicar el archivo llamado granja-de-orson-game.wpgm y ejecutar click derecho > Run As > Wollok Program.

    FOTO

    Si vemos dentro del archivo encontraremos un programa Wollok con un comentario explicando los controles del juego.

    FOTO

    Una vez que el programa se haya ejecutado se abrirá una nueva ventana con nuestro juego, así que a jugar. El alumno avanzará para llegar al objetivo del juego hasta que surja un error...

    FOTO

    .. y tenga que resolverlo para continuar.

    FOTO

    En el origen del problema se encuentra una explicación acerca de lo que debería hacer el juego. El alumno codificará dichos requerimientos y volverá a jugar para probarlo.

    FOTO

    A medida que avance, el juego liberará nuevos lugares y dirá nuevas pistas.

    FOTO

    Una vez que se haya completado el juego y ganado, éste terminará y se cerrará solo.
